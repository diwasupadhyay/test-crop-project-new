# Crop Price Predictor - Frame Sequence

This folder contains the frame sequence for the scroll-based hero animation.

## Structure

```
sequence/
├── frame_0001.png
├── frame_0002.png
├── frame_0003.png
├── ...
└── frame_0120.png
```

## Specifications

- **Total Frames**: 120 frames (for smooth 60fps animation)
- **Resolution**: 1920x1080 (Full HD)
- **Format**: PNG with transparency support
- **Content**: Animated crop growth/transformation sequence

## Recommended Content

1. **Frames 1-24**: Wheat field growing
2. **Frames 25-48**: Rice paddy visualization
3. **Frames 49-72**: Maize/Corn growth
4. **Frames 73-96**: Cotton field blooming
5. **Frames 97-120**: Sugarcane field

## Placeholder Generation

If you don't have actual frames, the application will generate placeholder frames with:
- Gradient backgrounds
- Animated crop emojis
- Particle effects

## Creating Custom Frames

You can create frames using:
1. **After Effects**: Export as PNG sequence
2. **Blender**: Render animation frames
3. **Premiere Pro**: Export frame sequence
4. **Online tools**: Lottie to PNG sequence converters

## Tips for Best Results

- Maintain consistent lighting across frames
- Use smooth transitions between crops
- Include subtle motion blur for realism
- Optimize file sizes for web performance
